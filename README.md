# Customer Behavior Analysis Project

Customer Behavior Analysis using Python, SQL, Power BI and Gamma.

## Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The objective is to clean and analyze customer data, identify purchasing patterns, generate business insights, and present findings through an interactive dashboard and presentation.

## Dataset

The dataset contains customer shopping information, including:

Customer ID
Age
Gender
Product Category
Purchase Amount
Location
Subscription Status
Shipping Type
Previous Purchases
Review Rating
Payment Method
Purchase Frequency

How to Run
Python Analysis
Open the Jupyter Notebook.
Install required libraries:

pip install pandas numpy matplotlib seaborn sqlalchemy pymysql

Run all notebook cells.
SQL Analysis
Import the dataset into MySQL, PostgreSQL, or SQL Server.
Execute the SQL queries provided in the project.
Power BI Dashboard
Open the .pbix file in Power BI Desktop.
Refresh the data source if required.
Explore the dashboard visuals and filters.
